Viktiga Teknologier

HTTP

Hypertext Transfer Protocol (HTTP) är det kommunikationsprotokoll som används för att överföra webbsidor på det globala informationsnätverket World Wide Web. Det ursprungliga syftet med HTTP var att möjliggöra överföringen av HTML-sidor från webbservrar till webbklienter. Utvecklingen av HTTP har koordinerats av World Wide Web Consortium (W3C) samt arbetsgrupper inom Internet Engineering Task Force, och nådde sin kulmination med publiceringen av flera RFC-dokument, med RFC 2616 (tidigare RFC 2068) som den mest centrala. Den definierar HTTP/1. 1, den version av HTTP som idag är mest utbredd.

SSL/TLS

Secure Sockets Layer (SSL) är ett säkerhetsprotokoll som utvecklades av Netscape för att kryptera kommunikationen mellan två enheter. Idag har SSL ersatts av Transport Layer Security (TLS) som åtgärdar säkerhetsbrister som fanns i SSL.

Cachning

Cachning innebär att en kopia av nyligen använda data temporärt lagras i ett snabbt minne för att kunna återanvändas vid senare tillfällen. Målet med cachning är att påskynda exekveringstiden genom att förhindra behovet av att hämta samma data igen via ett långsamt nätverk eller från ett långsammare minne.

Ett cacheminne är en hårdvarukomponent, ofta i form av statiskt RAM (SRAM), som finns i mikroprocessorer och används för att mellanlagra programkod eller data som nyligen har hämtats från det större, men långsammare arbetsminnet (vanligtvis dynamiskt minne, DRAM).
